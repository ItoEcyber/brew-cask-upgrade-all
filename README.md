# brew-cask-upgrade-all

# Requirements

- macOS
- ruby


# What is this doing

1. `brew update`
1. `brew upgrade`
1. check latest version of cask items. `uninstall --force` and `install` that have update version.
1. `brew cleanup`
1. `brew cask cleanup`

# usage

```
$ brew-cask-upgrade-all [-f]

-f : force upgrade that newest version is 'latest'
```
